# CRUD API
A CRUD API + Client powered by MEVN stack

## Setup Development Machine
1. Clone the repo `git clone git@github.com:emongelo/crud-app.git`

2. `cd crud-app`

3. Open client
```
cd client
npm run dev
```

4. Open server
```
cd server
npm start
```

5. Open `http://localhost:3000` in browser
